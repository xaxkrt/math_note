1_平面直角坐标系的基本概念

  1. 平面直角坐标系: 两条互相垂直的数轴

    2. 三要素: 原点(0), 正方向(右&上), 单位长度("1")

  2. 一点 两轴 四象限
    1. 一点: 两条数轴的交点: 公共原点 O
    2. 两轴: 坐标轴
      • 横轴/x轴
      • 纵轴/y轴
    3. 四象限: 四个区域(第一象限&第二象限&第三象限&第四象限)
  3. 点的坐标: P(a, b) (a: 横坐标, b: 纵坐标)

2_点的特征之横纵坐标正负性

点与四大象限

  1. 点 A(x, y) 在第一象限, 则
    • x > 0
    • y > 0
  2. 点 B(x, y) 在第二象限, 则
    • x < 0
    • y > 0
  3. 点 C(x, y) 在第三象限, 则
    • x < 0
    • y < 0
  4. 点 D(x, y) 在第一象限, 则
    • x > 0
    • y < 0
  5. 点 E(x, y) 在x轴上, 则 y = 0
  6. 点 F(x, y) 在x轴上方, 则 y > 0
  7. 点 G(x, y) 在x轴下方, 则 y < 0
  8. 点 H(x, y) 在y轴上, 则 x = 0
  9. 点 I(x, y) 在y轴右方, 则 x > 0
  10. 点 J(x, y) 在y轴左方, 则 x < 0
  11. 点 K(x, y) 在坐标轴上, 则 x = 0 或 y = 0

3_点的特征之横纵坐标大小关系

点与象限的角平分线

  1. 点 A(x, y) 在第一/三象限的
    • 角平分线上, 则 x = y
    • 角平分线上方, 则 x < y
    • 角平分线下方, 则 x > y
  2. 点 A(x, y) 在第二/四象限的
    • 角平分线上, 则 x + y = 0
    • 角平分线上方, 则 x + y > 0
    • 角平分线下方, 则 x + y < 0

两点与坐标轴

  1. 若 A(xa, ya), B(xb, yb) 满足
    • AB // x轴, 则
      • xa ≠ xb
      • ya = yb
    • AB // y轴, 则
      • xa = xb
      • ya ≠ yb

4_线的特性之平行&距离&中点

平行于坐标轴的线

  1. 过点 A(a, b) 且平行于
    • x轴的直线为 y = b
    • y轴的直线为 x = a

线段长度

  1. 两点间的距离公式: A(x1, y1), B(x2, y2), C(x1, y2)
    • A, C 两点横坐标相同: AC = |y1 - y2|
    • B, C 两点纵坐标相同: BC = |x1 - x2|
    • A, B 两点距离公式: AB = ( x 1 − x 2 ) 2 + ( y 1 − y 2 ) 2

线段中点表示

  1. 若 A(x1, y1), B(x2, y2), 则 AB 中点 M( x 1 + x 2 2 , y 1 + y 2 2 )

5_点的平移

将点 A(x, y)

  • 向上平移 m 个单位, 得 A1(x, y + m) : 横坐标不变, 纵坐标变大
  • 向下平移 m 个单位, 得 A2(x, y - m) : 横坐标不变, 纵坐标变小
  • 向左平移 m 个单位, 得 A3(x - m, y) : 横坐标变小, 纵坐标不变
  • 向上平移 m 个单位, 得 A4(x + m, y) : 横坐标变大, 纵坐标不变

6_点的翻折之坐标轴对称

点 B(x, y)

  1. 关于x轴对称, 得 B1(x, -y) : 横坐标不变, 纵坐标变成相反数
  2. 关于y轴对称, 得 B2(-x, y) : 纵坐标不变, 横坐标变成相反数
  3. 关于原点对称, 得 B3(-x, -y) : 横, 纵坐标均变成相反数

7_点的翻折之普通线对称

点 C(x, y)

  • 关于 y = b 对称, 得 C1(x, 2b - y) : 横坐标不变, 纵坐标对称
  • 关于 x = a 对称, 得 C2(2a - x, y) : 纵坐标不变, 横坐标对称
  • 关于点 M(a, b) 对称, 得 C2(2a - x, 2b - y) : 横, 纵坐均标对称

点 D(x, y)

  • 关于 y = x 对称, 得 D1(y, x) : 横, 纵坐均互换
  • 关于 y = -x 对称, 得 D2(-y, -x) : 横, 纵坐均互换并变为相反数

8_点的旋转

题型 方法
旋转90° 画图, 数格子
旋转180° 转化为关于点的翻折问题

例: 将点 A(1, 3) 绕点 B(4, 2) 顺时针旋转90°得A1, 顺时针旋转180°得A2

类型: 周期问题

9_三角形面积问题

  1. 有一边与坐标轴平行或重合 : 该边作为底

    2. S△ABC = 1 2 |xB - xC||yA - yB|

    S△ABC = 1 2 |yA - yB||xA - xC|

  2. 与坐标轴无平行边 : 沿顶点补全

    3. S = S总 - S1 - S2 - S3

10_等腰三角形存在性问题

常见题型: 已知 点A, B, △ABC 为等腰三角形, 求坐标轴上 点C 的可能性
解题思路: 分3类讨论

  1. AB = AC (AB 为腰):
    以 A 为圆心, AB 为半径作圆, 交 x轴 于 C1, C2, 交 y轴 于 C3, C4
  2. AB = BC (AB 为腰):
    以 B 为圆心, AB 为半径作圆, 交 x轴 于 C5, C6, 交 y轴 于 C7, C8
  3. AC = BC (AB 为底):
    作 线段AB 的中垂线, 交 x轴 于 C9, 交 y轴 于 C10
